# 🌍 Kora-Tutor

## Product Vision
Preserving the Nama language, one child at a time. Kora is an AI-powered language learning platform built for the Khoekhoē people.

## 🚀 Quick Start
```bash
npm install
npm run dev
```

## 🛠️ Stack
- **Frontend:** Next.js 15 (App Router)
- **Database/Auth:** Firebase
- **AI:** Google Gemini 1.5 Flash
- **Audio:** Firebase Storage + WaveSurfer.js
